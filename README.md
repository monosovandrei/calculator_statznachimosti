# Калькулятор достоверности конверсии

Веб-калькулятор для оценки статистической достоверности наблюдаемой конверсии. Помогает определить, насколько можно доверять текущим показателям при заданном количестве диалогов, и спрогнозировать, как изменится точность при увеличении выборки.

![Скриншот калькулятора](screenshot.png)

## Локальный запуск без Docker

Откройте `src/index.html` в браузере.

## Запуск в Docker

```bash
docker-compose up -d
# доступно на http://localhost:8080
```

## Сборка вручную

```bash
docker build -t conversion-calculator .
docker run -d -p 8080:80 conversion-calculator
```

## Структура проекта

```
.
├── src/
│   ├── index.html      # Разметка калькулятора
│   ├── style.css       # Стили, темы, адаптив
│   └── app.js          # Логика расчётов
├── Dockerfile
├── docker-compose.yml
├── nginx.conf          # Конфигурация nginx
├── .dockerignore
├── .gitignore
└── README.md
```

## Кастомизация

- **Тексты вердиктов и пороги** — `src/app.js`, функция `update()`, блок условий по `margin`
- **Цветовая палитра** — `src/style.css`, CSS-переменные в `:root` и `@media (prefers-color-scheme: dark)`
- **Целевые точности в брейкпоинтах** — `src/app.js`, массив `targets`
